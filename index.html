The Story

Beyond the Code

I came to software through an unconventional path - Physics first, then Social Work, then a Java codebase at an aerospace company in the UAE. What looked like a zigzag was actually the same thing each time: figuring out how things work, and making them work better.

My internship at AMC Aerospace Technologies was my first real exposure to enterprise software : ERP systems, Java, JSF, and what it actually means to write code that a business depends on. I picked it up fast, with minimal hand-holding, and left with a much sharper sense of how backend systems are structured and why they're built that way.

Outside of Java, I work mostly in Python , building things with SQL, REST APIs, and databases. The Physics degree didn't teach me to code, but it taught me how to think through problems methodically until they break open. That habit has been more useful than I expected.
